# YTClassicDarkTheme

A standalone tweak that restores YouTube’s classic gray dark theme and adds optional OLED / Custom color modes.

## Features

- Restores the classic gray dark theme
- Supports OLED black mode
- Supports Custom Color mode with preset colors
- Optional OLED Keyboard mode
- Works inside YouTube in-app settings
- Works without depending on YouGroupSettings

## Theme Modes

- Off
- Classic Gray
- OLED
- Custom

## Custom Color Presets

- Slate Gray
- Warm Gray
- Midnight Blue
- Deep Purple
- Olive Dark
- Charcoal
- None

## Settings

Inside YouTube settings, this tweak provides:

- Mode
- OLED Keyboard
- Custom Color

## Important Notes

- Theme changes apply after fully restarting YouTube
- OLED Keyboard changes also apply after fully restarting YouTube
- Selecting a Custom Color automatically sets Mode to Custom
- Selecting None clears the saved custom color and sets Mode to Off

## Compatibility

- Rootless compatible
- Designed for the YouTube app
- Can coexist with YouGroupSettings
- Does not require YouGroupSettings to function

## Installation

Install the package, then fully close and reopen YouTube.
