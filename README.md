# YTClassicDarkTheme

A standalone tweak that restores YouTube’s classic gray dark theme.

## Features

- Restores the classic gray dark mode background
- Works as a standalone tweak
- Rootless compatible
- Designed for the YouTube app

## Notes

- This tweak targets the YouTube app only
- This package uses ElleKit

## Installation

Install the package, then fully close and reopen YouTube.

## Compatibility

Tested as a standalone classic gray dark theme tweak for YouTube.
